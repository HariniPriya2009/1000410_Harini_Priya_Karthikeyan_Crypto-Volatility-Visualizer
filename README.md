Crypto Volatility Visualizer - Real Data Version

Dataset

File: Crypto_data.crdownload

Records: 637,114 data points

Date Range: January 1, 2012 to March 18, 2013

Granularity: Minute-level data

Columns: Timestamp (Unix), Open, High, Low, Close, Volume

 How to Use

live link: https://1000410-harini-priya-karthikeyan-crypto-volatility-visualizer.streamlit.app/
 
 Dashboard Features

 Time Range Control (Sidebar)

Select how many days of data to view: 7, 15, 30, 60, or 90 days

Default: 30 days

Data Granularity (Sidebar)

Daily: Aggregated from minute data to daily OHLCV (recommended)

Hourly: Aggregated hourly for more detail

Raw: Original minute-level data (can be overwhelming)

Data Source (Sidebar)

Use default dataset: Uses Crypto_data.crdownload file
